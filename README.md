# ticketing-frontend

Frontend del sistema de venta de entradas. Ver la documentación completa en el [README raíz del proyecto](../README.md).

## Comandos

```bash
npm run dev              # servidor de desarrollo → http://localhost:5173
npm run build            # build de producción
npm run test:e2e         # tests Cypress headless
npm run test:e2e:open    # GUI de Cypress
```

## Stack

React 19 · TypeScript · Vite · TailwindCSS v4 · Auth0 · React Router · Cypress
